# KMAutomator
A mainly command line interface-based keyboard and mouse input emulator built on the IntelliJ IDEA platform.

MouseLocation "MouseXPos, MouseYPos"
LeftMouseClick
LeftMouseDown
LeftMouseUp
RightMouseClick
RightMouseDown
RightMouseUp
Type "Text"
KeyDown "Key"
KeyUp "Key"
Loop "LoopTimes" ()
Delay "Seconds"  

note: "Loop" is work in progress
